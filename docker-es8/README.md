## Docker for Elasticsearch8
  
### 1. 버전
- Elasticsearch 8.1.1
  
### 2. 구성
- [docker-compose.yml (.env 참조)](https://github.com/kimheonseung/elasticsearch8/blob/master/docker-es8/docker-compose.yml)
  
### 3. 실행
```shell
# Linux
./init.sh

# Windows
.\init.bat
```

### 특이사항
- Windows의 경우 WSL 사용 여부와 무관, File Sharing 허용 필요
