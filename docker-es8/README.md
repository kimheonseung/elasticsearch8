## Docker for Elasticsearch8
  
### 1. Version
- Elasticsearch 8.1.1
  
### 2. Options
- [docker-compose.yml (with .env)](https://github.com/kimheonseung/elasticsearch8/blob/master/docker-es8/docker-compose.yml)
  
### 3. How to run
```shell
# Linux
./init.sh

# Windows
.\init.bat
```
